# Purpose

This document defines the complete visual identity of Project Cortex.

It is the authoritative specification for all visual decisions made during implementation.

If another document conflicts with this one, this document takes precedence for visual appearance.

The objective is to create a premium technological experience that feels intentionally engineered rather than artistically improvised.

Every rendered pixel should reinforce the perception of precision, intelligence, and craftsmanship.

---

# Design Philosophy

The interface should communicate

Engineering

before

Science Fiction.

The objective is not to imitate cyberpunk aesthetics.

The objective is to communicate computational intelligence through industrial design.

Imagine that Apple hired NVIDIA's graphics engineers to design the operating system for an artificial intelligence.

That is the emotional target.

# Color Philosophy

The interface exists in darkness.

Darkness allows information to become visible.

Color is reserved exclusively for information.

Black is structure.

Gray is machinery.

Color is computation.

Nothing else should emit light.

# Surface Detail

Every surface should reward inspection.

From a distance

The silhouette dominates.

At medium range

Circuit routing becomes visible.

Close inspection reveals

vias

chips

pads

micro traces

bevels

engraving

surface variation

Nothing should ever appear flat.

---

# Shape Language

Everything follows engineering principles.

Rounded where required.

Sharp where functional.

Consistent chamfers.

Large uninterrupted surfaces are discouraged.

Mechanical segmentation is encouraged.

---

# Edge Philosophy

All important edges should catch light.

Micro bevels should exist wherever possible.

Perfectly sharp edges rarely exist in manufactured objects.

---

# Lighting Philosophy

The object should never be fully illuminated.

Darkness creates mystery.

Light reveals information.

Lighting should feel cinematic.

Preferred setup

HDRI

↓

Soft hemisphere fill

↓

Strong rim light

↓

Cool key light

↓

Warm secondary bounce

↓

Selective bloom

↓

Ambient occlusion

↓

Screen space reflections (optional)

---

# Bloom

Bloom should exist only where energy exists.

Bloom should never wash out detail.

Target appearance

Apple product launch

not

cheap glow effect.

---

# Reflection Philosophy

Reflections should be subtle.

Only metallic regions reflect strongly.

Glass reflects softly.

Graphite barely reflects.

Reflection is a material property.

Not decoration.

---

# Typography

Font style

Modern

Geometric

Highly readable

Avoid decorative fonts.

Typography should support the visualization.

Never compete with it.

# Base Platform

The brain should never float in empty space.

# Camera

The camera should feel intentional.

Never perfectly orthographic.

Preferred focal length

35–50mm equivalent.

Slight perspective.

Slight downward angle.

Comfortable viewing distance.

Every pixel must justify its existence.
